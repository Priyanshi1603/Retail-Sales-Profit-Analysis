# 📊 Retail Sales & Profit Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing retail sales data to identify business trends, evaluate profitability, and generate actionable insights. The project includes data cleaning, KPI development, business analysis, and dashboard creation using Power BI.

The objective of this project is to transform raw retail transaction data into meaningful business intelligence that supports data-driven decision-making.

---

## 🎯 Project Objectives

* Clean and prepare retail sales data for analysis
* Analyze overall sales and profitability performance
* Identify high-performing regions and products
* Evaluate customer segment profitability
* Track monthly sales trends
* Create an interactive Power BI dashboard
* Generate business insights from the dataset

## 🧹 Data Cleaning Process

The following preprocessing steps were performed before analysis:

* Checked the dataset for duplicate records
* Identified missing values in the Product Base Margin column
* Replaced missing values using mean imputation
* Standardized Order Date and Ship Date formats
* Verified numeric columns for accurate calculations
* Prepared the dataset for business analysis and visualization

## 🔍 Business Insights

### Highest Sales Region

* West Region generated the highest sales revenue.

### Most Profitable Product Category

* Technology emerged as the most profitable category.

### Top Revenue Generating Products

* Global Troy Executive Leather Low-Back Tilter
* Polycom ViewStation ISDN Videoconferencing Unit
* Canon PC940 Copier

### Most Profitable Customer Segment

* Corporate customers generated the highest profit contribution.

### Monthly Sales Trend

* Sales remained strong during the beginning and end of the year.
* Mid-year sales were comparatively lower.
* Seasonal sales fluctuations were observed throughout the year.
